Scittle nREPL Demo

Interactive Browser REPL

This page demonstrates:

  • Scittle running ClojureScript in the browser
  • nREPL-over-WebSocket connection to Babashka
  • Live code evaluation from your editor

Instructions:

  1. Run bb dev in this directory
  2. Connect your editor to nREPL on port 1339
  3. Edit and evaluate playground.cljs
